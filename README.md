# YUN随记

一个随记APP，提供随时记录、添加习惯、添加待办、添加笔记等功能。



功能简介：

### 主题管理

#### 实现功能：

- 二阶主题内容管理
- 所有记录都可关联到主题下

#### TODO：

- 多级主题



## 记录

#### 实现功能：

- 自定义记录条目
- 添加记录

#### TODO：

- 添加更多数据类型（正则、图片、附件等）
- 添加记录统计
- 记录不一定关联主题

- 根据模板添加自定义统计页面（可用 web 实现）





## 习惯

#### 实现功能：

- 添加习惯提醒
